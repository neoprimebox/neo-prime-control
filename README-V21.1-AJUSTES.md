# Neo Prime Control V21.1 — Ajustes

- Data de novos pedidos calculada pelo horário local do computador, evitando iniciar com o dia seguinte.
- Ícone do calendário destacado em todos os campos de data.
- Campo CPF do cliente adicionado ao formulário de pedidos e persistido com os dados do pedido/cliente.
- Busca global limpa automaticamente ao trocar de tela, permanecendo apenas na tela em que a pesquisa foi feita.
