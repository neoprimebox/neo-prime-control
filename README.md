# Neo Prime Control v14.0

Versão focada na operação atual de dropshipping da Neo Prime Box, sem exigir FBA ou controle de estoque neste momento.

## Principais ajustes da v14

- Tela de Configurações reorganizada.
- Removidos campos sem função como Cor principal e Status da loja.
- Painel de Operação com dados do sistema, banco e quantidades cadastradas.
- Nova área de Contas de Compra para cadastrar contas usadas na Shopee, TikTok Shop, Mercado Livre etc.
- Cadastro de pedidos com campo Conta usada na compra.
- Removido o campo Marketplace da compra, pois o Fornecedor já indica a origem da compra.
- Mantido foco em vendas Amazon e compras via fornecedores/marketplaces.
- Estrutura preparada para futura evolução com estoque próprio/FBA, sem adicionar complexidade agora.

## Observação

As contas de compra são salvas localmente no navegador nesta versão. Os pedidos continuam seguindo a estrutura atual do sistema/Supabase, usando o campo de observações extras para manter compatibilidade.
