Escolha como importar

1. Importar por JSON recomendado

Cole aqui o JSON estruturado do pedido. O custo Amazon pode ficar 0 e você preenche depois.

2. Importar Excel/CSV

Use sua planilha de vendas .xlsx ou CSV. A V12 trata delimitador ;, valores em R$, linhas vazias e endereço parcial.

3. Cadastro manual assistido

Use o formulário de pedidos quando estiver olhando o print da Amazon e preferir preencher campo a campo.

🤖 Análise automática por print

Integração IA/OCR não configurada. Por isso, o sistema não deve fingir que consegue ler o print automaticamente.

Use uma das opções acima: JSON, CSV ou preenchimento manual. Quando uma integração real for configurada, esta opção poderá ser ativada.
